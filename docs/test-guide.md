### 4. API 接口测试

#### 4.1 短片项目 API
```bash
# 获取项目列表
curl -H "Authorization: Bearer <token>" http://localhost:5000/api/shortfilm/projects

# 获取复刻项目
curl -H "Authorization: Bearer <token>" "http://localhost:5000/api/shortfilm/projects?sourceType=remake"

# 创建项目
curl -X POST -H "Authorization: Bearer <token>" -H "Content-Type: application/json" \
  -d '{"name":"测试项目","sourceType":"original"}' \
  http://localhost:5000/api/shortfilm/projects
```

#### 4.2 复刻 API
```bash
# 链接解析
curl -X POST -H "Authorization: Bearer <token>" -H "Content-Type: application/json" \
  -d '{"url":"https://example.com/video.mp4","projectId":"test-123"}' \
  http://localhost:5000/api/shortfilm/remake-link

# AI解析（需要先有sourceVideoKey）
curl -X POST -H "Authorization: Bearer <token>" \
  http://localhost:5000/api/shortfilm/remake-parse/<projectId>
```

### 6. 风险提示

1. **AI解析超时**：解析可能需要较长时间，已设置10分钟超时
2. **视频文件大小**：限制500MB以内
3. **链接解析**：依赖yt-dlp支持的平台
4. **积分消耗**：生成图片/视频会消耗用户积分
